# ember-rails-tdd
An Ember (frontend) &amp; Rails (backend) Application with the TDD Process.

## Prerequisites
 * **Backend**
  * ruby 2.2.1p85
  * rails 4.2.1
 * **Frontend**
  * node v0.12.2
  * npm 2.7.4
  * ember-cli 0.2.3
  * ember-cli-pretender 0.1.0
